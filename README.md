
**O Sistema de Gerenciamento de Biblioteca** 
Este sistema é projetado para automatizar e otimizar as operações essenciais de uma biblioteca, garantindo o controle eficiente do acervo, dos membros e das transações de empréstimo

**Requisitos**
● Bibliotecários e membros podem se logar e ter acesso ao sistema.​ 
● Bibliotecários podem ter o cargo de estagiário, atendente ou supervisor.
●​ Qualquer pessoa que queira fazer um empréstimo deve ser um membro.
●​ Usuários não devem ser excluídos, mas podem ter sua conta desativada, o que
impossibilita o acesso ao sistema.
●​ Apenas bibliotecários com cargo de supervisor podem gerenciar o cadastro de
bibliotecários.
●​ Apenas bibliotecários podem gerenciar o cadastro de membros, livros e
empréstimos.
●​ Cada membro pode visualizar o acervo da biblioteca e uma lista dos seus
empréstimos.
●​ A multa por empréstimo atrasado é de suspensão de 1 dia para cada dia
atrasado de cada livro.●​ O sistema não deve realizar empréstimos a membros que estejam cumprindo
suspensão.
